

# Journal - Frontend

A lightweight React + Vite + TypeScript frontend for the journal app. It allows users to register, log in, and manage journal entries categorized by topic or type.

## Tech Stack

- React
- Vite
- TypeScript
- Axios
- React Router
- React Redux
- Tailwind CSS

## Features

- User authentication
- Create and view journal entries
- Filter entries by category
- Responsive, clean UI


## Setup Instructions

1. **Clone the repository**

```
https://github.com/Volburaal/journal-frontend.git
cd journal-frontend
```

2. **Install Dependencies**

```
npm install
cd journal-backend
```

3. **Set Environment Variables**
```
VITE_API_BASE_URL : Backend URL
```

**Start** using ```npm run dev``` to run it locally
